  2. Type: npm run dev
  3. Wait for "ready on http://localhost:3000"

╔════════════════════════════════════════════════════════════════════════════╗

╚════════════════════════════════════════════════════════════════════════════╝

  1. Go to: http://localhost:3000/report
  2. You should see the report form with upload area

╔════════════════════════════════════════════════════════════════════════════╗
║ STEP 3: TEST GEOLOCATION (20 seconds)                                     ║
╚════════════════════════════════════════════════════════════════════════════╝

Look for the BLUE button that says: "🌍 Detect GPS"

  1. Click the button
  2. Browser will ask for permission (at top of page)
  3. Click "Allow" when prompted
  4. Wait 5-15 seconds for GPS to lock
  5. You should see: "Location detected! Latitude: X.XXXX, Longitude: Y.YYYY"

Result: ✅ Coordinates appear in the form

╔════════════════════════════════════════════════════════════════════════════╗
║ STEP 4: VERIFY MAP UPDATES (10 seconds)                                   ║
╚════════════════════════════════════════════════════════════════════════════╝

Scroll down to see:
  - LATITUDE and LONGITUDE fields (should have values)
  - MAP with blue marker at your location
  - Marker should be approximately where you are

Result: ✅ Map shows your location

╔════════════════════════════════════════════════════════════════════════════╗
║ STEP 5: TEST FORM SUBMISSION (1 minute)                                   ║
╚════════════════════════════════════════════════════════════════════════════╝

  1. Upload an image or video (or use the pre-filled data)
  2. Scroll to bottom
  3. Click blue "Submit Report to CivicPulse" button
  4. Form should submit successfully

Result: ✅ You're redirected to dashboard

╔════════════════════════════════════════════════════════════════════════════╗
║ IF SOMETHING GOES WRONG                                                    ║
╚════════════════════════════════════════════════════════════════════════════╝

Problem: "Detecting..." forever (won't finish)
  → Wait 20 seconds (timeout)
  → Check browser permission prompt (top of page)
  → Grant permission if asked

Problem: "Permission Denied" error
  → Click 🔒 Lock in address bar
  → Change Location to "Allow"
  → Refresh page (F5)
  → Try again

Problem: Map doesn't appear
  → Refresh page (F5)
  → Try "Detect GPS" again

Problem: Browser doesn't support
  → Use Chrome, Firefox, Safari, or Edge (not IE)
  → Update browser to latest version

Problem: Can't grant permission
  → Use MANUAL MAP SELECTION instead:
    1. Drag the blue marker to your location
    2. Or click on the map
    3. Coordinates update automatically
    4. Form still works 100%!

║ CHROME PERMISSION QUICK FIX                                                ║
╚════════════════════════════════════════════════════════════════════════════╝

If Chrome asks for permission but you accidentally clicked "Block":

  1. Click 🔒 LOCK icon in address bar (top-left)
  2. Find "Location" in the dropdown
  3. Change from "Block" to "Allow"
  4. Refresh page (F5)
  5. Try "Detect GPS" again

For best accuracy on mobile:

  1. Enable location services:
     • iPhone: Settings → Privacy → Location Services → ON
     • Android: Settings → Location → ON

  2. Go to: http://localhost:3000/report (or your Vercel URL)
  3. Tap "Detect GPS"
  4. Allow permission when asked
  5. Wait for location detection

Mobile GPS is more accurate outdoors!
